# 🎬 Jamilur's Watched

A personal anime watchlist web app where you can upload, rate, and organize anime posters.

---

## 🚀 Features

- 📤 Upload anime posters
- ⭐ 1–3 star rating system
- ✏️ Rename anime (click name)
- 🗑️ Delete anime
- 🔄 Drag & drop reordering
- 💾 Auto-save using localStorage
- 🎨 Clean Netflix-style UI with hover effects

---

## 🛠️ Built With

- React.js
- Tailwind CSS
- JavaScript (ES6)
- localStorage (for saving data)

---

## 📱 How it works

- All data is saved in your browser automatically
- No login required
- Works on both mobile and desktop
